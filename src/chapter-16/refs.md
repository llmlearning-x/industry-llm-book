## 研究笔记

### 电信行业大模型应用分层

1. **L1 基础客服**：RAG + 意图识别，落地率>55%
2. **L2 网络运维**：RAG + Agent + 规则引擎，落地率~25%
3. **L3 精准营销**：用户画像 + RAG + Agent，落地率~20%
4. **L4 网络自优化**：多模态 + 时序预测 + Agent，落地率<15%

### 多厂商设备集成要点

- 华为VRP、中兴ZXROS、爱立信CLI/ML、诺基亚OSS/CLI — 各厂商CLI语法差异显著
- 知识库必须按厂商隔离索引，防止命令串厂商
- 所有AI生成的操作命令必须经过独立的语法校验层

### 实时性架构设计

- 快路（7B模型）：P99 < 1s，处理实时告警关联
- 慢路（72B模型）：深度分析 + 知识库异步更新
- 降级预案：大模型异常时自动回退到基于规则的传统排障流程
